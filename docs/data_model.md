# Data Model

## Fact Table
**fct_orders** — represents each order placed on the marketplace.

## Dimensions
- **dim_customers** — customer data
- **dim_sellers** — seller data
- **dim_products** — product data
- **dim_payments** — payment method and value
- **dim_geolocation** — geographic location

## Business Questions
1. Which states purchase the most?
2. Which product categories sell the most?
3. Which months have the highest sales?
4. Which sellers have the worst ratings?
5. What is the average delivery time by state, and how often are orders delivered late?
